1. Регулярные выражения.

Дан пример вида: 'esdfd((esdf)(esdf' -> 'esdfd((esdf)'

  • Если в начале строки стоит текст, то его требуется вывести в строке.

  • Если в строке нашлась открывающаяся скобка то требуется вывести весь текст внутри нее до закрывающейся скобки

  • Если закрывающейся скобки не найдено то текст не выводим.

{% csrf_token %}

Ответ: {{ resultSearch }}

Алгоритм работы регулярного выражения:

Вся продукция на складе:

  • ^[a-zA-Z]* - в начале строки может находится какое то количество букв.

  • [(]*[a-zA-Z]*[)] - поиск осущетвляется между скобками каких то букв.

  • ([(]*[a-zA-Z]*[)])* - подобных вариантов может быть несколько.

pattern = '^[a-zA-Z]*([(]*[a-zA-Z]*[)])*'

resultSearch = re.search(pattern, str)

где ^[a-zA-Z]*([(]*[a-zA-Z]*[)])* наше регулярное выражение,

  • ^[a-zA-Z]* - в начале строки может находится какое то количество букв.

  • [(]*[a-zA-Z]*[)] - поиск осущетвляется между скобками каких то букв.

  • ([(]*[a-zA-Z]*[)])* - подобных вариантов может быть несколько.

# Без регулярного выражения
i = 0
buf = ''
char = ''
for elem in str:
   if i == 0 and elem != '(':
      char = char + elem
   elif elem == '(' and buf is None:
      buf = elem
   elif elem == '(' and buf is not None:
      buf = buf + elem
   elif elem == ')':
      char = char + buf + elem
      buf = ''
   else:
      buf = buf + elem
   i += 1
